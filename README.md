### Install dependencies

```
npm install
```

### Running in production

```
npm start
```

Runs on localhost:3000 by default but can be configured using the `PORT` environment variable.

### Running tests

```
npm test

# Watch repo
npm run test:watch
```

### Running in development

```
npm run dev
```

### Linting
```
npm run lint

# fix issues
npm run lint:fix
```
